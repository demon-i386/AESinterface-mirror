# Implementação AES C ++
#### Versão 0.0.1 (beta)

 @versão em teste !!

## Sobre

  AES é uma criptografia baseada em bloco,
  estamos trabalhando com blocos de 16 bits no momento.
  
## Instalação

  Para compilar, use `make install`, e então você pode executar `make run`.
